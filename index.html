How it works

Up and running
in three steps.

01
Install Ollama and pull a model

Ollama runs models locally on your machine. Pull any code model you like.

$ ollama pull qwen2.5-coder:7b
02
Open viber and pick a folder

Point viber at your project. It indexes the codebase in the background for semantic search.

03
Chat, plan, and edit

Ask questions or request changes. Review AI-generated edits and apply them with one click.
